# EDA_Assignment1.

Dataset Source: Kaggle – “Retail Product Dataset with Missing Values”

Number of Rows and Columns: 4,345 rows × 5 columns

Features (Columns):

Category: The product category (e.g., A, B, C, D, Unknown).
Price: The price of the product in the local currency.
Rating: Customer rating for the product (numeric, 1–5).
Stock: Stock status of the product (In Stock, Out of Stock, Unknown).
Discount: Discount applied to the product (percentage).

Purpose of Using this Dataset:
This dataset was selected to practice Exploratory Data Analysis (EDA), including handling missing values, duplicates, and incorrect formats. It allows analysis of pricing, ratings, discounts, and stock patterns across different product categories, providing insights useful for business decisions and sales strategy planning.
